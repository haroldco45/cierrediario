# 🏪 Cierre Diario

**Cierra la caja de tu negocio en 3 minutos, desde el celular y sin internet.**

PWA para tenderos y pequeños negocios de Caucasia y el Bajo Cauca. Cuando bajas las puertas, abres la app: anotas lo que entró, lo que salió, y la app te dice tu ganancia y si la caja cuadró.

## ✨ Funciones

- **Múltiples cajas**: Caja 1, mostrador, la del ayudante… cada una con su propio cierre.
- **Saldo inicial (base de caja)**: para saber cuánto debe haber al final.
- **Ingresos detallados**: ventas mostrador, domicilios, fiado recuperado, Nequi/Daviplata, otros.
- **Gastos detallados**: mercancía, servicios, arriendo, nómina, gasto personal, otros.
- **Cuadre de caja**: escribes el efectivo contado y la app dice si sobra, falta o cuadró.
- **Billete de ganancia**: la ganancia del día en una tarjeta estilo billete 💵.
- **Historial 30 días**: gráfica, mejor día, promedio diario y detalle por día/caja.
- **Resumen por WhatsApp**: envía el cierre formateado con un toque.
- **Copia de seguridad**: exporta e importa tus datos en un archivo `.json`.
- **100% local**: los datos viven solo en el teléfono del dueño. Sin servidores, sin internet.

## 🚀 Instalación

1. Sube `index.html` a cualquier hosting (GitHub Pages, Netlify, cPanel).
2. Abre la URL en el celular con Chrome.
3. Menú → **"Agregar a pantalla de inicio"**. Listo, queda como una app.

> La imagen OG (`og-cierre-diario.png`) va incluida en este repo — solo sube los 3 archivos juntos y WhatsApp mostrará la vista previa automáticamente.

## 🔒 Datos

La app no envía información a internet (Ley 1581/2012 — los datos financieros del negocio nunca salen del dispositivo del dueño). Se recomienda descargar una copia de seguridad semanal desde **Ajustes**.

## 🛠️ Tecnología

- HTML + CSS + JavaScript en un solo archivo
- `localStorage` para persistencia
- PWA con manifest embebido
- Hora y fecha de Colombia (UTC-5)
